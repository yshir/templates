# vite-ts

- [TypeScript](https://www.typescriptlang.org/)
- [vite](https://ja.vitejs.dev/)
- [React Router](https://reactrouter.com/en/main)
- [tailwindcss](https://tailwindcss.com/)

## Prerequisites

```sh
$ node -v
v20.10.0
```

```sh
$ npm -v

```

